# Router Admin — панель управления роутером (демо)

Эмулятор веб-интерфейса настройки роутера: вкладки (Настройка, WLAN5G, WLAN2.4G, EasyMesh), подгрузка контента, мастер настройки.

## Запуск

Приложение использует ES-модули и `fetch()` для загрузки HTML. **Открытие `index.html` через `file://` в браузере не сработает** — нужен локальный HTTP-сервер.

### Вариант 1: Python

```bash
cd router-admin
python3 -m http.server 8080
```

Откройте в браузере: **http://localhost:8080**

### Вариант 2: Node.js (npx)

```bash
cd router-admin
npx -y serve -l 8080
```

Откройте: **http://localhost:8080**

### Вариант 3: PHP

```bash
cd router-admin
php -S localhost:8080
```

---

После запуска сервера откройте указанный адрес — должны работать вкладки, сайдбар и подгрузка разделов (Мастер, WLAN 5G/2.4G, EasyMesh).
