# Эмуляторы прошивок роутеров — контекст проекта

## Обзор проекта

Коллекция веб-страниц с эмуляторами и инструкциями по настройке роутеров различных производителей (D-Link, TP-Link, Asus, Keenetic, Cudy, Tenda, Mercusys, Xiaomi, Huawei, Netis, SNR, Mikrotik и др.).

Проект предназначен для использования монтажниками и техническими специалистами для быстрой настройки оборудования.

## Структура проекта


## Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — кастомные свойства (CSS variables), анимации, flexbox/grid
- **Vanilla JavaScript** — ES6+ модули, fetch API
- **Font Awesome 5.15.3** — иконки
- **Google Fonts** — шрифт Ubuntu

## Ключевые возможности

### Тёмная/светлая тема
- Переключение через кнопку в хедере
- Сохранение предпочтения в `localStorage`
- CSS-переменные для цветовых схем

### Адаптивность
- Мобильная версия в `/mobile/`
- Grid-галерея с колонками (3 → 2 → 1)
- Уважение к `prefers-reduced-motion`

### Роутер-админ панель (`/router-admin/`)
- Вкладки: НАСТРОЙКА, WLAN5G, WLAN2.4G, EASY MESH, TCP/IP, IPV6, MULTIWAN
- Сайдбар с разделами: МАСТЕР, РЕЖИМ РАБОТЫ
- Подгрузка контента через ES-модули

## Запуск проекта

### Основной сайт
Просто откройте `index.html` в браузере (работает через `file://`).

### Мобильная версия
Откройте `mobile/index.html` в браузере.

### Router-admin панель
Требуется HTTP-сервер (используется `fetch()` и ES-модули):

```bash
# Python
cd router-admin
python3 -m http.server 8080

# Node.js
npx -y serve -l 8080

# PHP
php -S localhost:8080
```

Откройте: **http://localhost:8080**

## Соглашения по разработке

- **Кодировка:** UTF-8
- **Язык интерфейса:** русский
- **Стили:** BEM-подобная именование классов (`.emulators__item`, `.card-header`)
- **JS:** ES6+ модули, `const`/`let`, стрелочные функции
- **Доступность:** `aria-label`, `role`, фокус-стили
- **Изображения:** формат WebP с PNG-фоллбэком

## Примечания

- Проект не использует сборщики (webpack, vite) — чистый статический сайт
- Внешние зависимости подключаются через CDN (Font Awesome, Google Fonts)
- Все пути относительные (портативность)
